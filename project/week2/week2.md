좋아, 너가 올려준 문제 리스트를 보면 다양한 알고리즘 유형들이 섞여 있어.  
그래서 **자료구조 → 정렬 → 탐색 → 이분 탐색 → 투 포인터 → 그리디 → 분할정복 → 우선순위 큐 → 세그먼트 트리 → 스위핑**처럼 점진적으로 확장하는 흐름으로 학습하는 걸 추천할게.
